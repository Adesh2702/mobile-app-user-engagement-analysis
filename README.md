# 🔍 Key Insights

- Average session duration is approximately **30 minutes** across devices.
- **Subscription distribution is balanced** across Free, Trial, and Premium users.
- Churn risk is **distributed across all subscription types**, indicating churn risk is not isolated to one segment.
- Monthly logins show **consistent engagement with slight fluctuations** over time.
- User engagement is **similar across countries**, suggesting global usage consistency.
- Age groups show **similar session patterns**, indicating the app appeals to multiple demographics.
